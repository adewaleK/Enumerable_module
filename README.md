## About The Project

In this project, I recreated the following Ruby Methods for the Enumerables Module:
#my_each
#my_each_with_index
#my_select
#my_all?
#my_any?
#my_none?
#my_count
#my_map
#my_inject
#multiply_els

<br>
<br>
For more details about the Project and it's requirements go to <a href="https://www.theodinproject.com/courses/ruby-programming/lessons/advanced-building-blocks"> The Odin Project</a>
